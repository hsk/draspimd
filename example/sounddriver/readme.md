# サウンドドライバ

- psg01 PSGを鳴らしてみる
- psg02 PSGを直接操作してみる
- psg03 ノイズを鳴らしてみる
- vgm01 VGMファイルをダンプしてみる
- vgm02 メガドライブでVGMを再生してみる
- vgm03 効果音を鳴らしてみる
- vgm04 効果音がなったときだけBGMのパートの音を消す
